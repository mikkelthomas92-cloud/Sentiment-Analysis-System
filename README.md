# Sentiment Analysis System

## Description
This project performs sentiment analysis on movie reviews using Machine Learning.

## Dataset
IMDb 50K Movie Reviews Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run:
python sentiment_analysis.py
